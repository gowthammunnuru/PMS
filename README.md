# Welcome to Perform
**An online Peformance Management system for DDU**

## How to Start/Stop/Restart Perform

You must be root.

**Start**
```
systemctl start perform
```

**Stop**
```
systemctl stop perform
```

**Restart**

```
systemctl restart perform
```
